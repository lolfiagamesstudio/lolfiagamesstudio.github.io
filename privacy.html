Política de Privacidade

O jogo Coelho - Nem Pares é um produto do LOLFIA Games Studio (Luciano Oliveira).

1. Recolha de Dados

Não recolhemos informações pessoais que te identifiquem diretamente. No entanto, o jogo utiliza serviços de terceiros (Google AdMob) que podem recolher dados usados para te identificar através de um ID de publicidade.

2. Publicidade (AdMob)

Este jogo contém anúncios. O Google AdMob pode utilizar o teu ID de publicidade do Android para mostrar anúncios relevantes e para fins de análise.

3. Segurança

A segurança dos teus dados é importante para nós, mas lembra-te que nenhum método de transmissão pela internet é 100% seguro.

4. Contacto

Se tiveres dúvidas, contacta-nos: lolfiagamesstudio@gmail.com
